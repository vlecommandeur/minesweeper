# Minesweeper
## _Using Rust and WASM_

✨ A fun minesweeper to be improved. ✨

## Tech

- [Rust]
- [WASM]

## Installation

It requires [wasm-pack] and [serve] to run.

```sh
wasm-pack build --target web
serve
```
